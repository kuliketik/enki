Project Enki
------------

# Tentang

Enki adalah dewa pengetahuan dan kebijaksanaan (wisdom) bangsa Mesopotania,
proyek ini bertujuan untuk mendapatkan pengetahuan mengenai engagement
(like, comment) pada media online. 


# media online target

- detik.com

- tribunews.com

- tirto.id

- theconversation.com

- merdeka

- kompas.com
